# clickassist

A click assistant program that simulates touch screen events.

I wrote this to play games requiring a touch screen.

## Requirements

Requires PySide6.

Currently only works on KDE Wayland.

## Usage

```
python -m clickassist
```

Then click Record to bind a key to a mouse click. Press Start to hide the program to the system tray.
